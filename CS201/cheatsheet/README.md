# Cheatsheet

This cheatsheet is fairly comprehensive — almost encyclopedia-like. However, compared to the CS101 one, I don't find it as efficient to look up on the fly during exams. The main reason: the CS101 cheatsheet was hand-edited from the instructor's lecture notes, so I'm much more familiar with its layout and content. AI-generated cheatsheets are thorough and quick to produce, but it takes time for a person to build familiarity with them.

Still, this was a worthwhile experiment — a collaborative project with Claude Code. I had Claude Code systematically study the instructor's markdown lecture notes, then customize requirements around "benchmarking against LeetCode Top 100." Claude Code responded to these personalized requirements quite precisely.
