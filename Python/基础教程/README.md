# Python 基础教程

本目录包含Python基础知识的学习资料。

## 内容概览

### 1. 变量和数据类型
- 数字（整数、浮点数）
- 字符串
- 列表（List）
- 元组（Tuple）
- 字典（Dictionary）
- 集合（Set）

### 2. 控制流程
- if/elif/else 条件判断
- for 循环
- while 循环
- break 和 continue

### 3. 函数
- 函数定义和调用
- 参数传递
- 返回值
- Lambda 函数
- 作用域

### 4. 文件操作
- 读取文件
- 写入文件
- 文件上下文管理器

### 5. 异常处理
- try/except
- 自定义异常
- finally 语句

## 学习路径

建议按照以下顺序学习：
1. 先掌握基本数据类型
2. 理解控制流程
3. 学习函数的使用
4. 实践文件操作
5. 了解异常处理机制
